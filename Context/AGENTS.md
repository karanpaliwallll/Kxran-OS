# Context Routing

Use this folder for durable context that should improve future sessions.

Read order:

1. `agent-brief.md` for the short always-load context.
2. `retrieval-map.md` to decide what else to open.
3. `memory.md` for stable facts and preferences.
4. `corrections.md` for behavior changes and known mistakes.

Do not store raw session history here. Put history in `Sessions/` and promote only durable lessons into this folder.
