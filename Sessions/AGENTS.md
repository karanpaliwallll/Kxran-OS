# Sessions Routing

Store each AI-agent session once under `Sessions/YYYY/MM/YYYY-MM-DD-title.md`.

## Rules

- Use `Sessions/_index.md` for global lookup.
- Use `Sessions/active.md` for current handoff.
- Link project-specific sessions from `Projects/<project>/sessions.md`.
- Do not duplicate full sessions into project folders.
- Do not load all sessions by default.

## Session Notes Are Archives

When a fact should guide future behavior, promote it into `Context/`, `Projects/<project>/status.md`, or `.project/single-source-of-truth.md`.
